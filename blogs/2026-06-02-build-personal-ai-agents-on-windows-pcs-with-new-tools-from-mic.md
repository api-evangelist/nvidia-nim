NVIDIA and Microsoft unveiled new tools enabling developers to build secure, on-device AI agents for Windows platforms, including Microsoft eXecution Containers for sandboxing agents and NVIDIA RTX Spark hardware with 1 petaflop AI capability. The collaboration delivers 2x faster inference speeds and addresses growing demand for local AI development with enhanced security and native app integration. The announcement provides optimized inference backends for agent deployment on personal hardware.
